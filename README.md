# Portfolio Website

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS.

![Portfolio Website](https://img.shields.io/badge/Next.js-14-black)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## ✨ Features

- ⚡ **Fast & Optimized** - Built with Next.js 14 and App Router
- 🎨 **Modern UI** - Clean and responsive design with Tailwind CSS
- 📱 **Mobile Responsive** - Looks great on all devices
- 🎯 **SEO Optimized** - Meta tags and semantic HTML
- 🚀 **Performance** - Optimized fonts with next/font and image optimization
- 📝 **TypeScript** - Type-safe code for better development experience

---

## 🛠️ Built With

- [Next.js 14](https://nextjs.org/) - React Framework
- [TypeScript](https://www.typescriptlang.org/) - Type Safety
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Geist Font](https://vercel.com/font) - Typography

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js 18.x or higher
- npm / yarn / pnpm / bun

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yadavxprakhar/portfolio-website.git
   cd portfolio-website


2. **Install Dependencies**

    Bash
    npm install

3. **Run the Devlopment Server**

   Bash
   npm run dev
4. **Open Your Browser**

   Navigate to http://localhost:3000 to see the website.

---

## 📁 Project Structure

```
portfolio-website/
├── app/                # Next.js app directory
│   ├── page.tsx       # Home page
│   ├── layout.tsx     # Root layout
│   └── ...
├── components/        # React components
├── public/           # Static assets
├── styles/           # Global styles
└── ...
```
---

## 🎨 Customization

Edit app/page.tsx to modify the main page content
Update components in the components/ directory
Modify styles in tailwind.config.ts for theme customization
Add your images to the public/ folder
📦 Build for Production
Bash

npm run build
npm start
